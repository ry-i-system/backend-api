# backend-api

### repo clone
```
git clone https://github.com/ry-i-system/backend-api.git
```

### change directory
```
cd backend-api
```

### pipenv exec
```
pipenv shell
```

### pipenv module install
```
pipenv install
```

### web server start
```
python manage.py runserver
```

### Rest API login
http://localhost:8000/api/v1/rest-auth/login/
